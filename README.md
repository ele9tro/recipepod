# [Recipe Pod](https://recipepod.herokuapp.com)  
  レシピ共有アプリケーション  
  
  
## 概要  
  自身の料理画像にレシピを添えて投稿できます。   
  他者の料理画像を閲覧でき、美味しい料理のレシピを得られます。

## 制作背景
既存のレシピ共有サイトは文字での情報量が多いのでもっと直感的に料理を選べた方が楽しいのはないかと考え、  
写真から興味を持った料理を選択し、レシピを参照することができるようするため。
